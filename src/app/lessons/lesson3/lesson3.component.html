Install i3 + tuning status bar

i3 + status bar

Install all required packages, not under root. You should do this command from created user, non-root user.

pacman -S xorg-init xorg-server
yaourt -S ttf-font-awesome

Intalling i3wm and input number 2

pacman -S i3wm

Add to /etc/X11/xinit/xinitrc following line, an get something next as on picture

exec i3

For start we should input next command

startx

Now we go edit status-bar for that we need create ".xinitrc" in our home directory, in my case is following directory "home/banan", so we do next commands

cd /etc/X11/xinit/
cp xinitrc /home/banan/
cd /etc/i3/
cp config /home/banan/

You can download code of "/etc/i3/config" below, you cam add list of starting programms on starting i3 in the end of i3cconfig, as on exemple below:

exec program1

File i3status.conf is responsible for display status bar in bottom of your screen

Install thunar

yaourt -S thunar
