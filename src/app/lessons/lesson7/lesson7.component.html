Archive Manager

Installing Archive Manager

yaourt -S file-roller

For access to archive(zip) open from console If archive manager doesn't display contain of archive, then change extenson to zip and add following command for apply to file "file"

sudo chmod 777 file
sudo thunar .
change extentsion file to zip
open...

For extract archive "file.rar":

yaourt -S unrar
unrar x file.rar

TimeZone

Setting languages and locale. You can toggle language by ctr_alt.

localectl --no-convert set-x11-keymap us,ru,de "" "" grp:ctrl_alt_toggle
timedatectl set-timezone Europe/Kiev

Setting locale for displaying right UTF-8, for example: display russian charaacters in archive-manager, for more information click Here

Uncomment in /etc/locale.gen following lines

de_DE.UTF-8
en_US.UTF-8
ru_UA.UTF-8
Generate: locale-gen
Checking locale: locale -a
localectl set-locale LANG=en_US.UTF-8, LANG=ru_UA.UTF-8, LANG=de_DE.UTF-8
