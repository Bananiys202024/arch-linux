Automatically connect wifi on boot

You should use netctl. It is the Arch Linux built in network management system. If it isn't already installed (it should be though), run

pacman -Syu netctl
cd /etc/netctl/
cp /etc/netctl/example/wireless-wpa /etc/netctl/somedescriptivename

then edit /etc/netctl/somedescriptivename

systemctl enable netctl-auto@wlan0.service
