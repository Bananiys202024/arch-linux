USB-stick

Mounting usb-stick with hands

Mount usb-stick

yaourt -S ntfs-3g
lsblk
sudo mkdir /mnt/my_usb/
sudo mount /dev/sdd /mnt/my_usb
cd /mnt/my_usb

Automatically mounting usb-stick

Mount usb-stick automatically

yaourt -S ntfs-3g
yaourt -S thunar-volman yaourt -S udiskie

Configure thunar-volame After installing the plugin "thunar-volman"

1. Launch Thunar and go to Edit -> Preferences
2. Under the 'Advanced' tab, check 'Enable Volume Management'
3. Click configure and check the following items:
Mount removable drives when hot-plugged.
Mount removable media when inserted.

Execute magic command for automatically connecting usb-stick

udiskie -2 -s &
