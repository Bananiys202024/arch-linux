Install Arch linux

Step 1. Connecting wi-fi

First of all we need unlock wifi, for connecting to internet

rfkill unblock all

Find wi-fi. You should input next command and then choose wifi for connecting. Don't use in wifi this character "-"

wifi-menu

Step 2. Partionating disk

Partionating. Im my case, "sda" is main my disk, disk of computer.

gdisk /dev/sda

Then create new table

o

Then do all following action, as on image.

You can print following command to revise changes

lsblk

Then you should create encrypted part of disk. In my case "sda2" is my second disk

cryptsetup -y luksFormat /dev/sda2

Go open our created encrypted part of disk

cryptsetup open /dev/sda2 cryptroot

Create file system on ecnrypted part of disk

mkfs.ext4 /dev/mapper/cryptroot

Then simple do following commands:

mount /dev/mapper/cryptroot /mnt
cd /mnt
mkdir boot
mkfs.vfat /dev/sda1
mount /dev/sda1 /mnt/boot
pacstrap /mnt

Step 3. Install

Go to installed arch

arch-chroot /mnt

Installing all standart files

bootctl install

Step 3.

Intall vim editor

pacman -S vim

Intall packages for working wifi after installation

pacman -S dialect netctl networkmanager

For adding to core of support ecrypt , we should add line "encrypt" in file "/etc/mkinitcpio.conf" as shown below

Then go to /boot/loader and edit following file "loader.conf" with command:

vim loader.conf
